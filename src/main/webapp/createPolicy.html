You are about to delete a record in the database. Only select this option if the entire record is incorrect. Do not select this option when a policy is eliminated (e.g. an export ban has been lifted). In that case, create a new record using the "Add Policy" option and indicate that the policy has been eliminated.

Please provide a justification why this record has to be deleted. This justification is preferably a link to a government website or another reliable source. You can also upload a pdf version of the legal document or source document.
Note that the record will be deleted from the database once the justification has been checked.
