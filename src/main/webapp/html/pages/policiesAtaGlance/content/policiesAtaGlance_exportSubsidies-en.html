Export Subsidies

The AMIS policy database contains information on export subsidies starting from 1995 for the AMIS countries. Data on budgetary outlays and quantities (both commitments and notifications) are recorded for the 4 AMIS commodities (wheat, maize, rice and soybean). Export subsidies apply to groups of commodities, which can be composed of
  • commodities that all belong to the same AMIS commodity class (e.g. Maize), or
  • commodities that belong to a mixed AMIS commodity class (e.g. Maize & Wheat & Rice).
The composition of these groups of commodities (called shared groups) can be found in the file Shared Groups list posted on the Statistical Notes page.
The data are sourced from the WTO database on export subsidies. The link to this database is listed on the Statistical Notes page.

Select one country and one commodity class.




The graph displays export subsidies for the selected commodity class in the selected country in terms of quantity and budgetary outlay. Both the commitments and notifications are displayed. Zero values are also displayed and can be seen by hovering over the X-axis. Values are aggregated in countries with export subsidies for more than one group of commodities in a specific commodity class. Commitments and notification categories can be removed from the graph by deselecting them from the legend in the graph.
