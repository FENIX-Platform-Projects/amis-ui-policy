HS Code
Enter the most detailed HS code as specified in the source document. This field should be completed for trade policies and should be empty for domestic policies
HS Version
Select the HS version associated with the HS code. This field should be completed for trade policies (for measures starting in 2012 or later, please select HS2012) and should be empty for domestic policies
Description
Enter the Harmonized Commodity Description associated with the HS code and HS version or – if not available – provide the description of the commodity as specified in the source document
Short Description
Create a shorter version of the description that is maximum 255 characters long and that only contains the relevant parts of the description. If the description is already less than 255 characters, you can copy the description into the "short description" field
